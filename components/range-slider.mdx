---
title: RangeSlider
description: Two-handle range slider for selecting a value range with min/max constraints.
---

## Usage

```tsx
import { RangeSlider } from '@kivora/react';

function Demo() {
  const [range, setRange] = useState<[number, number]>([20, 80]);
  
  return (
    <RangeSlider 
      label="Price range" 
      value={range}
      onChange={setRange}
      min={0}
      max={100}
      showValues
    />
  );
}
```

## Props

<ParamField path="value" type="[number, number]">
  Controlled value as a tuple `[low, high]`.
</ParamField>

<ParamField path="defaultValue" type="[number, number]" default="[0, 100]">
  Default value for uncontrolled mode.
</ParamField>

<ParamField path="onChange" type="(value: [number, number]) => void">
  Called when either handle moves. Receives the new range tuple.
</ParamField>

<ParamField path="min" type="number" default="0">
  Minimum value of the slider range.
</ParamField>

<ParamField path="max" type="number" default="100">
  Maximum value of the slider range.
</ParamField>

<ParamField path="step" type="number" default="1">
  Granularity of value changes.
</ParamField>

<ParamField path="label" type="string">
  Label text displayed above the slider.
</ParamField>

<ParamField path="showValues" type="boolean" default="false">
  Displays the current range values (e.g., "20 – 80").
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the slider and prevents interaction.
</ParamField>

<ParamField path="color" type="'primary' | 'success' | 'warning' | 'error'" default="'primary'">
  Color theme for the active segment and thumbs.
</ParamField>

<ParamField path="minRange" type="number" default="0">
  Minimum distance required between low and high values.
</ParamField>

<ParamField path="marks" type="Array<{ value: number; label?: string }>">
  Optional mark indicators to display below the slider.
</ParamField>

<ParamField path="className" type="string" default="''">
  Additional CSS classes to apply to the container.
</ParamField>

### Price Filter Example

```tsx
function PriceFilter() {
  const [range, setRange] = useState<[number, number]>([10, 500]);
  
  return (
    <div>
      <RangeSlider
        label="Price"
        value={range}
```

## Notes

- Two independent handles control low and high values
- Active segment between handles shows selected range
- Handles have dynamic z-index - clicked handle moves to front
- Click on track moves nearest handle to that position
- `minRange` prevents handles from getting too close
- Values automatically clamped to enforce minRange constraint
- Both handles use native range inputs for accessibility
- Marks display below with optional labels
- Smooth transitions and hover effects
- Values display as "low – high" format when showValues is true
